# Archive Directory

This directory contains archived development artifacts that are not part of the public release.

## reddit-inspiration-2025-01/

Contains early development artifacts from scenario creation process. These files were used for inspiration during initial scenario development but are not part of the published benchmark.

**Note:** All scenarios in the published benchmark are researcher-generated fiction based on AARP/NAC statistics and peer-reviewed research. No real caregiver data, Reddit posts, or personally identifiable information were used in the final scenarios.

**Contents:**
- `caregivers_raw.json` - Raw data (archived for internal reference only)
- `caregivers_clean.json` - Processed data (archived for internal reference only)
- `caregivers_processed.json` - Processed templates (archived for internal reference only)
- `data_extraction.py` - Data extraction script (archived for internal reference only)
- `scenario_templates.json` - Early templates (archived for internal reference only)

These files are preserved for historical reference but should not be included in any public releases or distributions.
