# Archive

This directory holds historical or superseded material kept for provenance.

Nothing here is part of the active public benchmark contract.

## Current layout

- `archive/docs/` — superseded public docs and solution logs
- `archive/scripts/` — one-off or legacy helper scripts
- `archive/internal/evals/remediation_2026-03-31/` — frozen-corpus remediation bundle, inventories, and old review UI
- `archive/internal/evals/verifier/` — archived verifier memos and tranche outputs
- `archive/internal/plans/` — one-off planning documents
- `archive/benchmark/`, `archive/scenarios/` — older benchmark materials kept for provenance

If something is still part of day-to-day benchmark operation, it should not live here.
